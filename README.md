# AsteroidsExtreme

![](asteroids.gif)

Small Asteroids rip-off I made a few years ago. Made with Unity.

The code here was written in 2016, so expect a little spaghetti code.
Made with Unity. I did not include the entire project, only the scripts and sprites.

I included a Windows 32-bit build. Please note that during the time I was working on this, I was messing around
with customization, so the colors are a little out-of-whack. It's been so long since I worked on it, I am not
going to bother changing it. Here for demonstration purposes.
